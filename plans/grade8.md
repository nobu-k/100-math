# 中学2年（Grade 8）問題計画

## カリキュラム概要

中学2年では、式の計算（単項式と多項式の加減乗除）、連立二元一次方程式、平面図形の性質（平行線と角、多角形の角）、三角形の合同条件と証明、平行四辺形の性質と条件、一次関数（y = ax + b のグラフ・式・表）、二元一次方程式のグラフ、確率（場合の数と確率の計算）を扱う。

〔用語・記号〕 単項式、多項式、次数、同類項、二元一次方程式、連立方程式、対頂角、同位角、錯角、内角、外角、定義、証明、逆、反例、変化の割合、傾き、切片、確率

## A. 数と式

### A1. 式の計算

**カリキュラム要件:**
- 簡単な多項式の加法・減法
- 単項式の乗法・除法、および式の値
- 文字式を利用して数量や数量の関係を表し説明すること

**問題タイプ:**

#### A1-1. 多項式の加減
- 「(3x + 2y) + (5x − 4y) = □」→ 8x − 2y
- 「(4a − 3b + 1) − (2a + b − 5) = □」→ 2a − 4b + 6

**UI設計:**
- 式 + 回答欄（各項の係数を入力）
- 12問程度
- 設定: 2変数 / 3項以上 / 括弧の有無

**実装メモ:**
- 項の種類: x項、y項、定数項（or a, b, c）
- 同類項をまとめる処理
- 係数は−9〜+9の整数
- 回答は各項の係数を分けて入力（例: x の係数、y の係数、定数項）

#### A1-2. 単項式の乗除
- 「3a × 4b = □」→ 12ab
- 「(−2x)² × 3y = □」→ 12x²y
- 「12a²b ÷ 4a = □」→ 3ab
- 「(−6xy) ÷ 2x × 3y = □」→ −9y²

**UI設計:**
- 式 + 回答欄
- 12問程度
- 設定: 乗法のみ / 除法のみ / 混合

**実装メモ:**
- 文字の指数を追跡して整理
- 除法は割り切れるペアを逆算で生成
- 答えの形式: 係数 + 文字部分（アルファベット順）

#### A1-3. 式の値（多項式）
- 「x = 3, y = −2 のとき、2x² − 3xy + y の値は？」
  → 2(9) − 3(3)(−2) + (−2) = 18 + 18 − 2 = 34

**UI設計:**
- 式 + 代入値 + 回答欄
- 10問程度

**実装メモ:**
- 中1の式の値の発展版
- 2変数の代入、累乗を含む
- 答えが−50〜+50の範囲に収まるよう調整

#### A1-4. 式による説明
- 「連続する2つの偶数の和が偶数になることを式を使って説明しなさい」
- 「2桁の正の整数の十の位と一の位を入れ替えた数との差が9の倍数であることを示しなさい」

**UI設計:**
- 将来課題（記述式は自動採点困難）
- 穴埋め形式の誘導で部分的に対応可能
  - 「連続する2つの偶数を 2n, □ とおくと、和は □ + □ = □」

### A2. 連立方程式

**カリキュラム要件:**
- 二元一次方程式とその解の意味
- 連立方程式の必要性と解の意味
- 連立方程式を解くこと（加減法・代入法）
- 連立方程式を具体的な場面で活用すること

**問題タイプ:**

#### A2-1. 連立方程式（加減法）
- 「2x + 3y = 7, 4x − 3y = 5 → x = □, y = □」→ x = 2, y = 1
- 「3x + 2y = 12, x + 2y = 8 → x = □, y = □」→ x = 2, y = 3

**UI設計:**
- 連立方程式の表示 + x, y の2つの回答欄
- 12問程度
- 設定: 係数の範囲、一方を消去しやすい形 / 両方に係数をかける必要がある形

**実装メモ:**
- 解 (x, y) を整数で先に決め（−5〜+5）、2つの式を逆算で構成
- 係数は−6〜+6の整数
- 加減法で一方がすぐ消える問題と、係数を揃える必要がある問題を混ぜる

#### A2-2. 連立方程式（代入法）
- 「y = 2x − 1, 3x + y = 9 → x = □, y = □」→ x = 2, y = 3
- 「x = 3y + 2, 2x − y = 9 → x = □, y = □」→ x = 5, y = 1

**UI設計:**
- 一方が y = ... or x = ... の形の連立方程式
- 10問程度
- 設定: 代入法向きの形に整えた問題

**実装メモ:**
- 一方の式を「y = ax + b」形式で生成
- 代入後の方程式が整数解を持つよう調整

#### A2-3. 連立方程式（小数・分数の係数）
- 「0.3x + 0.2y = 1.4, x − y = 2 → x = □, y = □」
- 「x/2 + y/3 = 2, x − y = 1 → x = □, y = □」

**UI設計:**
- 12問程度
- 設定: 小数の係数 / 分数の係数

**実装メモ:**
- 小数: 10倍して整数化
- 分数: 分母の最小公倍数をかけて整数化
- 最終的に整数解になるよう逆算

## B. 図形

### B1. 平行線と角

**カリキュラム要件:**
- 対頂角の性質
- 平行線の同位角・錯角の性質
- 三角形の内角の和が180°であること
- 多角形の内角の和・外角の和

**問題タイプ:**

#### B1-1. 角度を求める（平行線と角）
- 「2直線が平行のとき、∠x の大きさを求めなさい」（図付き）
- 対頂角: 「∠a = 65° のとき ∠b = □°」→ 65°
- 同位角・錯角の利用

**UI設計:**
- SVGで平行線と交線を描画し、角度をマーク
- 1つの角度が既知、他を求める
- 10問程度
- 設定: 対頂角 / 同位角 / 錯角 / 混合

**実装メモ:**
- 角度は 30°〜150° の整数値
- SVG描画: 2本の平行線 + 1〜2本の交線
- 角のマーク（弧記号）とラベル

#### B1-2. 多角形の角
- 「五角形の内角の和は？」→ 540°
- 「正八角形の一つの内角は？」→ 135°
- 「多角形の外角の和は？」→ 360°
- 「内角の和が 1080° の多角形は何角形？」→ 八角形

**UI設計:**
- 文章題 + 回答欄
- 10問程度
- 設定: 内角の和 / 正多角形の一つの内角 / 外角の和 / n を求める

**実装メモ:**
- 内角の和 = (n − 2) × 180°
- 正多角形の一つの内角 = (n − 2) × 180° / n
- 外角の和 = 360°（常に定数）
- n = 3〜12 の範囲

#### B1-3. 三角形の角度
- 「三角形の2つの角が 50° と 70° のとき、残りの角は？」→ 60°
- 「三角形の外角の定理: 外角 = 隣り合わない2つの内角の和」
- 「∠A = 40°, ∠B = 65° のとき、∠C の外角は？」→ 105°

**UI設計:**
- 三角形のSVG描画 + 角度のラベル
- 8〜10問
- 設定: 内角を求める / 外角を求める

**実装メモ:**
- 内角の和 = 180°
- 外角 = 180° − 内角 = 他の2つの内角の和
- 角度は整数、鈍角三角形も含む

### B2. 三角形と四角形

**カリキュラム要件:**
- 三角形の合同条件
- 証明の必要性と意味、証明の進め方
- 二等辺三角形・正三角形の性質
- 直角三角形の合同条件
- 平行四辺形の性質と条件
- 長方形・ひし形・正方形と平行四辺形の関係

**問題タイプ:**

#### B2-1. 合同条件の選択
- 「△ABC ≡ △DEF を証明するために、必要な条件は次のうちどれ？」
- 3つの合同条件（三辺相等、二辺夾角相等、二角夾辺相等）の選択

**UI設計:**
- 図形のSVG描画 + 既知の条件 + 選択肢
- 8問程度
- 設定: 合同条件の判定 / 証明の穴埋め

**実装メモ:**
- 図形の条件をランダムに設定
- 合同条件のパターンを3つ + 直角三角形用2つ
- 将来的に証明の穴埋め問題に拡張

#### B2-2. 平行四辺形の性質
- 「平行四辺形ABCDで、AB = 8cm, BC = 5cm のとき CD = □cm」→ 8
- 「∠A = 110° のとき ∠B = □°」→ 70°
- 「対角線の交点をOとすると AO = □ CO」→ 等しい

**UI設計:**
- 平行四辺形のSVG描画 + 寸法
- 10問程度
- 設定: 辺の長さ / 角度 / 対角線

**実装メモ:**
- 平行四辺形の性質:
  - 対辺が等しい（AB = CD, BC = DA）
  - 対角が等しい（∠A = ∠C, ∠B = ∠D）
  - 対角線が互いに中点で交わる（AO = CO, BO = DO）
  - 隣り合う角の和が180°

## C. 関数

### C1. 一次関数

**カリキュラム要件:**
- 一次関数 y = ax + b の理解
- 変化の割合の意味
- 一次関数のグラフの特徴（傾きと切片）
- 二元一次方程式のグラフが直線であること
- 一次関数を活用して問題を解決すること

**問題タイプ:**

#### C1-1. 一次関数の式を求める
- 「傾き 2、切片 −3 の一次関数の式は？」→ y = 2x − 3
- 「変化の割合が −3 で、x = 2 のとき y = 1。式を求めなさい」→ y = −3x + 7
- 「2点 (1, 3) と (3, 7) を通る直線の式は？」→ y = 2x + 1

**UI設計:**
- 条件 + 回答欄（a と b を入力、y = ax + b）
- 12問程度
- 設定: 傾き+切片 / 傾き+1点 / 2点から求める

**実装メモ:**
- 傾き a: −5〜+5 の整数（0を除く）、または 1/2, −1/3 などの簡単な分数
- 切片 b: −10〜+10 の整数
- 2点から求める場合: 差分で傾きを出し、代入で切片を出す
- 答えの a, b が整数になる問題を中心に

#### C1-2. 変化の割合
- 「y = 3x − 2 の変化の割合は？」→ 3
- 「x が 1 から 4 まで増加するとき y は 5 から 14 に増加した。変化の割合は？」→ 3
- 一次関数では変化の割合 = 傾き（一定）

**UI設計:**
- 文章題 or 式 + 回答欄
- 8問程度
- 設定: 式から / 値の変化から / 表から

**実装メモ:**
- 変化の割合 = (yの増加量) / (xの増加量) = 傾き
- 一次関数では常に一定であることの理解

#### C1-3. グラフの特徴
- 「y = −2x + 3 のグラフの傾きは □、切片は □」
- 「グラフが右下がりの一次関数はどれ？」（複数の式から選択）
- 「y = 2x − 1 と y = −x + 5 の交点の座標は？」→ (2, 3)

**UI設計:**
- 式 or グラフのSVG描画 + 回答欄
- 10問程度
- 設定: 傾き・切片の判定 / グラフの向き / 交点

**実装メモ:**
- 交点: 連立方程式を解く（2x − 1 = −x + 5 → 3x = 6 → x = 2）
- SVGで座標平面にグラフを描画
- グリッド線とラベル付き

#### C1-4. 二元一次方程式のグラフ
- 「2x + 3y = 6 のグラフを描きなさい」
- 「2x + 3y = 6 のグラフの x 切片は □、y 切片は □」

**UI設計:**
- 式 + x切片・y切片の回答欄
- 8問程度

**実装メモ:**
- ax + by = c → y = −(a/b)x + c/b の変換
- x切片: y = 0 を代入 → x = c/a
- y切片: x = 0 を代入 → y = c/b
- a, b, c を整数にして切片が整数 or 簡単な分数になるよう調整

## D. データの活用

### D1. 確率

**カリキュラム要件:**
- 確率の必要性と意味
- 簡単な場合の確率の求め方
- 確率を用いて不確定な事象をとらえ説明すること

**問題タイプ:**

#### D1-1. 基本的な確率
- 「1つのさいころを投げて、3の倍数が出る確率は？」→ 2/6 = 1/3
- 「赤玉3個、白玉5個の袋から1個取り出して赤玉の確率は？」→ 3/8
- 「1枚の硬貨を2回投げて2回とも表が出る確率は？」→ 1/4

**UI設計:**
- 文章題 + 分数形式の回答欄
- 10問程度
- 設定: さいころ / カード / 玉 / 硬貨

**実装メモ:**
- 場合の数を求めて確率を計算
- 答えは既約分数で
- 分子・分母の回答欄を分けて入力

#### D1-2. 2つのさいころの確率
- 「2つのさいころを同時に投げて、和が7になる確率は？」→ 6/36 = 1/6
- 「2つのさいころを投げて、積が偶数になる確率は？」→ 27/36 = 3/4
- 表（6×6）を使った数え上げ

**UI設計:**
- 文章題 + 分数形式の回答欄
- 回答表示時に6×6の表を見せて該当マスをハイライト
- 8問程度
- 設定: 和 / 差 / 積 / 大小

**実装メモ:**
- 6×6 = 36通りの全列挙
- 条件に合うマスを数えて確率を計算
- 表のSVG描画で視覚的に確認

#### D1-3. 樹形図を使った確率
- 「A, B, C, D の4人から2人を選ぶとき、Aが選ばれる確率は？」→ 3/6 = 1/2
- 「3枚のカード 1, 2, 3 から2枚選んで2桁の数を作る。30以上になる確率は？」→ 2/6 = 1/3

**UI設計:**
- 文章題 + 分数形式の回答欄
- 回答表示時に樹形図をSVGで表示
- 8問程度
- 設定: 要素数（3〜5）

**実装メモ:**
- 順列・組み合わせの全列挙
- 6年生の場合の数の発展（確率の概念が追加）
- 樹形図のSVG描画
