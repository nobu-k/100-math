# 第2学年（Grade 2）問題計画

## カリキュラム概要

第2学年では、4位数までの数、2位数の筆算（加減）、乗法（九九）、簡単な分数、基本図形（三角形・四角形・箱）、長さ・かさの単位、時刻と時間、表とグラフを扱う。

---

## A. 数と計算

### A1. 数の構成と表し方（4位数まで）

**カリキュラム要件:**
- 同じ大きさの集まりにまとめて数える・分類して数える
- 4位数まで（千の位まで）の十進位取り記数法
- 数を十や百を単位として見る（数の相対的な大きさ）
- 一つの数を積として見る
- 簡単な分数（1/2, 1/4 など）
- 1万についても扱う（内容の取扱い）

**問題タイプ:**

#### A1-1. 大きな数の読み書き
- 数字を漢数字で書く、漢数字を数字で書く
- 「三百四十二」→「342」、「1058」→「千五十八」
- 位ごとに分解：「342 = 300 + 40 + 2」

**UI設計:**
- 左に問題、右に回答欄の2カラム形式
- 位取り表（千・百・十・一）を補助的に表示するオプション
- 10問程度

**実装メモ:**
- 漢数字⇔アラビア数字の変換ロジック
- 設定: 範囲（〜100、〜1000、〜10000）

#### A1-2. 数の大小・順序
- 不等号で比較：「523 □ 532」
- 3〜4個の数を小さい順・大きい順に並べる

**UI設計:**
- 比較問題: 1行1問で□に＞＜＝を入れる
- 並べ替え問題: 数を横に並べ、下に回答欄
- 設定: 数の範囲（2位数、3位数、4位数）

### A2. 加法及び減法（筆算）

**カリキュラム要件:**
- 2位数の加法・減法の筆算
- 簡単な3位数の加減
- 交換法則・結合法則
- 加法と減法の相互関係
- ( ) や □ を用いた式

**問題タイプ:**

#### A2-1. 2位数の筆算（たし算・ひき算）
- **既存機能**: Hissan で対応済み
- 桁数設定を2桁に固定して使用
- 繰り上がり・繰り下がりの有無を設定可能

#### A2-2. 3位数の簡単な加減
- **既存機能**: Hissan で対応済み（3桁設定）

#### A2-3. 加減の虫食い算
- 筆算形式で一部の数字が□になっている
- 例: □3 + 2□ = 58

**UI設計:**
- 筆算レイアウト（既存のHissan表示を流用）で□セルを空欄にする
- 回答時に□に数字を埋める
- 6〜8問

**実装メモ:**
- 完成した筆算を生成してからランダムに1〜2マスを□に置き換え
- 繰り上がり・繰り下がりが絡む位置の□は難易度が高い
- 設定: □の数（1個、2個）、繰り上がりの有無

### A3. 乗法（かけ算・九九）

**カリキュラム要件:**
- 乗法の意味（1つ分の数 × いくつ分）
- 乗法九九（1×1〜9×9）
- 乗法の簡単な性質（交換法則など）
- 簡単な2位数×1位数

**問題タイプ:**

#### A3-1. 九九の練習（百マス計算）
- **既存機能**: Grid100 の乗法で対応済み
- 9×9マスで九九の大量演習

**追加検討:**
- 特定の段だけに絞った練習モード（例: 7の段、8の段のみ）
- 順番バラバラの九九カード形式

#### A3-2. 九九の穴埋め
- 「7 × □ = 56」「□ × 6 = 48」
- 九九表の一部を□にして埋める

**UI設計:**
- 横書きの式で□を埋める形式（10〜15問）
- 九九表形式で一部セルを空白にするモードもあり
- 設定: 段の指定、□の位置（かけられる数、かける数、答え）

**実装メモ:**
- 九九の組み合わせからランダムに出題
- 重複排除

#### A3-3. 簡単な2位数×1位数
- 12×3、20×4 など
- **既存機能**: Hissan の乗法で桁数を制限して対応可能

## B. 図形

**カリキュラム要件:**
- 三角形・四角形を知る
- 正方形・長方形・直角三角形
- 箱の形（正方形・長方形の面で構成）

**問題タイプ:**

#### B2-1. 図形の名前・性質
- 図形を見て名前を答える、辺や頂点の数を答える
- **対応方針:** 図形の描画はSVGで可能だが、問題の多様性が限られる。将来課題。

## C. 測定

**カリキュラム要件:**
- 長さの単位: mm, cm, m
- かさの単位: mL, dL, L
- 測定の意味
- 時刻と時間（日、時、分）

**問題タイプ:**

#### C2-1. 単位の換算
- 「3cm 5mm = □mm」「120cm = □m □cm」
- 「2L 3dL = □dL」「1500mL = □L □mL」

**UI設計:**
- 1行1問で□に数値を記入
- 単位ごとのセクション分け（長さ / かさ）
- 10問程度
- 設定: 単位の種類、難易度（2単位間の換算、3単位の混合）

**実装メモ:**
- 単位間の変換係数テーブル
- 繰り上がりを含む換算（例: 180cm → 1m 80cm）

#### C2-2. 時刻と時間
- 「8時30分の20分後は何時何分？」
- 「9時から10時15分まで何時間何分？」
- 時計の図を見て時刻を読む（1年生の発展）

**UI設計:**
- 文章形式の問題と回答欄
- 時計のSVG図を添えるオプション
- 8〜10問
- 設定: 「○時ちょうど」「○時○分」「正午をまたぐ」

**実装メモ:**
- 分の繰り上がり（例: 8:50 + 30分 = 9:20）を含む問題生成
- 日をまたぐ問題は含めない

## D. データの活用

**カリキュラム要件:**
- 簡単な表やグラフの読み取り・作成
- データを分類整理

**問題タイプ:**

#### D2-1. 表の読み取り
- 簡単な集計表から数値を読み取る
- 「りんごは何個ですか？」「一番多い果物は？」

**UI設計:**
- HTML tableで集計表を表示し、設問を下に
- 5問程度

**実装メモ:**
- カテゴリ名と数値をランダム生成
- 合計の計算も含む
- 将来課題として棒グラフの描画・読み取りも検討

## 既存機能との対応表

| カリキュラム項目 | 既存機能 | 追加が必要な機能 |
|---|---|---|
| 2位数の加減筆算 | Hissan（+−） | 対応済み |
| 3位数の加減 | Hissan（+−） | 桁数設定で対応可 |
| 九九 | Grid100（×） | 段指定フィルタ、穴埋め形式 |
| 2位数×1位数 | Hissan（×） | 桁数設定で対応可 |
| 大きな数の読み書き | なし | **新規作成** |
| 数の大小比較（4位数） | なし | **新規作成**（1年生の拡張） |
| 加減の虫食い算 | なし | **新規作成** |
| 九九の穴埋め | なし | **新規作成** |
| 単位の換算（長さ・かさ）| なし | **新規作成** |
| 時刻と時間の計算 | なし | **新規作成** |
| 表の読み取り | なし | **新規作成** |
| 簡単な分数 | なし | 将来課題（1/2, 1/4 の概念） |

## 実装優先度

1. **高**: 九九の穴埋め（九九の定着は2年生最重要）
2. **高**: 加減の虫食い算（計算力の応用、筆算の理解深化）
3. **中**: 単位の換算（長さ・かさ）
4. **中**: 時刻と時間の計算
5. **中**: 大きな数の読み書き
6. **低**: 表の読み取り
7. **低**: 図形の識別
